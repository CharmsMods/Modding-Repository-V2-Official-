Venge.io Modding Repository

A professional-grade, browser-based platform for modding Venge.io—your complete ecosystem for creating, editing, organizing, and deploying custom game assets. Built with pure ES6+ JavaScript and the HTML5 Canvas API, it delivers over 1,200 assets and robust tools for texture, audio, UI, and environment modifications—no external servers required.

Core Architecture

  • Frontend Vanilla JavaScript (ES6+), modular design
  • Rendering & Editing HTML5 Canvas 2D Context for real-time image processing and non-destructive workflow
  • Storage SessionStorage, Blob URLs, IndexedDB-ready
  • Export JSZip 3.10.1 + FileSaver.js 2.0.5
  • UI CSS Grid/Flexbox + Font Awesome 6 icons
  • Extension Chrome Manifest V2 with webRequest interception

Key Interface Features

Asset Management

  • Real-time, case-insensitive filtering across filenames, folder IDs, types
  • Responsive card-grid with lazy loading & smooth CSS transitions
  • Alphabetical sorting, visual state indicators, folder-ID grouping
  • Fullscreen preview with zoom, resolution display, filename overlay

Professional Asset Editor

  • Live Canvas editing preserves originals; automatic sizing & memory-safe Blob handling
  • Saturation control (0–200%) via luminance-based algorithm
  • Dual modes:
  • - Modify Mode for existing textures
  • - Create Mode for new assets (1×1 to 4096×4096 px, color picker, solid fills)
  • Automatic JPEG⇄PNG conversion; metadata retention; quality-preserving compression

Bulk Operations Suite

  • Multi-selection with visual indicators & MP3 exclusion
  • Batch tools: saturation processing, texture generation, upload/conversion, resizing, exclusion toggles
  • Real-time progress bars, console logging, async Promise.all() processing, error isolation

Import/Export System

  • JSON configs with image hashing, metadata preservation, incremental updates, Base64 support
  • Client Export (game-ready) & Browser Export (extension-ready) modes
  • ZIP packages with DEFLATE, selective asset exclusion
  • Schema validation, corrupted import recovery, state persistence

Browser Extension & Performance

Browser Extension Integration

  • Manifest V2: webRequest/webRequestBlocking, access to *.venge.io/*
  • Background Script: directory scanning, URL pattern matching, recursive asset coverage
  • Resource Types: images, audio, 3D models, styles, text—intercepted & redirected to local assets with CORS handling

Performance & Optimization

  • Memory: Blob URL lifecycle management, automatic revocation, efficient garbage collection
  • Lazy Loading: Intersection Observer; priority queuing; thumbnail caching
  • Processing: Reusable Canvas contexts; hardware acceleration; debounced O(1) search

Extensibility & Security

  • Modular Codebase: Separate modules for asset lists, editor, bulk ops, import/export, converters, UI popups
  • Adding Assets: Drop in mod-assets/, update registry .txt, include metadata
  • Client-Side Only: All processing in browser, no external network; minimal extension permissions; input validation; sandboxed execution
